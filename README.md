# Dracula for [cmder](http://cmder.net/)

> A dark theme for [cmder](http://cmder.net/).

![Screenshot](https://raw.githubusercontent.com/oguhpereira/Dracula-cmder/master/cmdericon.PNG)

## Install

* Open your cmder
* Click Win+Alt+P
* Click in Import
* Select the Dracula.xml
* Save Settings
